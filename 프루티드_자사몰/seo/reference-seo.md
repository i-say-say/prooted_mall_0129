### 공통 패턴
1. **브랜드명**이 항상 포함
2. **구분자**: `-` 또는 `|` 사용
3. **후행 요소**: 슬로건, 포지셔닝, 또는 브랜드 설명

## 페이지 유형별 Meta 구조 패턴

### 1. Home (홈페이지)

**Title 패턴**
```
{브랜드명} | {핵심 가치/슬로건}
```

**Description 패턴**
```
{브랜드 포지셔닝}. {핵심 기준/특징 나열}. {신뢰 요소}
```

**예시 (추정)**
- Growus: "탈라소테라피 헤어&스파케어 브랜드. 두피와 모발을 위한 프리미엄 케어."
- Refilled: "본질을 채우는 헤어 케어. 과학적으로 검증된 탈모 솔루션."

---

### 2. Brand / About (브랜드 소개)

**Title 패턴**
```
{브랜드 철학 키워드} | {브랜드명}
또는
{브랜드명} 소개 | {브랜드명} 공식몰
```

**Description 패턴**
```
{브랜드 미션}. {핵심 가치 1~2개}. {차별화 포인트}
```

---

### 3. Category / Shop (카테고리)

**Title 패턴**
```
{카테고리명} | {브랜드명}
또는
{카테고리명} - {브랜드명} 공식몰
```

**Description 패턴**
```
{브랜드명}의 {카테고리명} 제품입니다. {카테고리 특징}. {공통 USP}
```

**예시**
- "탈모케어 샴푸 | Growus"
- "두피고민별 - 닥터포헤어 공식몰"

---

### 4. Product Detail (상품 상세)

**Title 패턴**
```
{제품명} | {브랜드명}
또는
{제품명} - {핵심 효능} | {브랜드명}
```

**Description 패턴**
```
{제품명}. {핵심 성분} 함유. {주요 효능 1~2개}. {인증/FREE 정보}
```

**예시**
- "씨솔트 테라피 두피 스케일러 | Growus"
- "폴리젠 샴푸 - 탈모 완화 | 닥터포헤어"

---

### 5. Story / Ingredient (성분 스토리)

**Title 패턴**
```
{원료명} {효능 키워드} | {브랜드명}
또는
{원료명}의 비밀 | {브랜드명} Superfood Story
```

**Description 패턴**
```
{원료명}이 {효능}에 좋은 이유. {원료 특징}. {관련 제품 언급}
```

---

### 6. Community / Review (커뮤니티)

**Title 패턴**
```
{페이지 유형} | {브랜드명}
```

**Description 패턴**
```
{브랜드명} {페이지 유형}입니다. {페이지 설명}
```

**예시**
- "고객 리뷰 | Growus"
- "자주 묻는 질문 FAQ | 닥터포헤어"

---

## SEO Best Practices (공통)

### Title 작성 규칙
1. **길이**: 50~60자 이내 (한글 기준)
2. **핵심 키워드**: 앞쪽에 배치
3. **브랜드명**: 뒤쪽에 배치 (인지도 낮을 경우)
4. **구분자**: `|` 또는 `-` 통일

### Description 작성 규칙
1. **길이**: 120~150자 이내 (한글 기준)
2. **첫 문장**: 핵심 정보 (검색 결과에 노출)
3. **CTA 암시**: "지금 확인하세요" 등
4. **키워드**: 자연스럽게 1~2개 포함

### 피해야 할 것
- 키워드 과다 나열 (스팸 판정)
- 모든 페이지 동일한 description
- 과장된 표현 ("최고의", "완벽한")
