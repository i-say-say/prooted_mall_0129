# 프루티드 자사몰 네비게이션 설계

## 헤더 네비게이션

### 데스크톱 (PC)

```
┌─────────────────────────────────────────────────────────────────────┐
│  🎁 첫구매 최대 60% 할인 + 5,000원 쿠폰                    [X 닫기] │  ← 프로모션 바 (Mintree 참고)
├─────────────────────────────────────────────────────────────────────┤
│ [LOGO]    SHOP  SUPERFOOD STORY  BRAND  COMMUNITY    🔍 👤 🛒      │
└─────────────────────────────────────────────────────────────────────┘
```

### 프로모션 바 (Top Bar) - Mintree 참고
- 헤더 최상단 고정
- 첫구매/회원가입 혜택 상시 노출
- X 버튼으로 닫기 가능 (쿠키로 상태 저장)

### 메인 메뉴 구조

| 메뉴 | 서브메뉴 | 설명 |
|------|----------|------|
| **SHOP** | 전체상품, BEST, NEW, 샴푸, 트리트먼트, 헤어에센스, 헤어팩, 세트/기획 | 드롭다운 메가메뉴 |
| **SUPERFOOD STORY** | 퍼플콤플렉스, 아사이베리, 블루베리, 성분사전 | 브랜드 차별화 콘텐츠 |
| **BRAND** | 프루티드 소개, 브랜드 철학, 클린뷰티 | 브랜드 스토리 |
| **COMMUNITY** | 공지사항, 이벤트, 리뷰, FAQ | 고객 소통 |

## 플로팅/고정 요소

- TOP 버튼 (스크롤 시 표시)

- 장바구니 미니 모달: 상품 추가 시 표시

