## 1. 메인 페이지 (Home)

### 섹션 구성 (상단→하단)

| 순서 | 섹션명 | 설명 | 참고 브랜드 |

| 5 | 고민별 추천 | 탈모/손상모/두피 탭 형식 | ViveLab |

### 상품 카드 정보
- 상품 이미지 (호버 시 2번째 이미지)
- 뱃지: BEST, NEW, SALE
- 상품명
- **상품 태그** (한 줄 키워드) ← Mintree 참고
- 가격 (할인가 표시)
- 평점 + 리뷰 수
- 찜하기 버튼

### 상품 태그 예시 (Mintree 참고)
```
퍼플부스터 샴푸: "탈모케어, 두피탄력, 비건인증, 설페이트FREE"
그린디톡스 샴푸: "두피진정, 피지케어, 비건인증, 민감두피"
바나나에그 트리트먼트: "손상모케어, 단백질영양, 즉각효과"
```

### 필터 옵션
- 고민별: 탈모케어, 손상모, 두피케어, 보습
- 가격대: ~2만원, 2~4만원, 4만원~
- 평점: 4점 이상

### 정렬 옵션
- 인기순 (기본)
- 최신순
- 가격 낮은순
- 가격 높은순
- 리뷰 많은순

## 3. 상품 상세 페이지 (PDP)

### 상단 영역

```
┌────────────────────────────────────────────────────────────────┐
│  ┌─────────────────────┐  ┌────────────────────────────────┐  │
│  │                     │  │  퍼플부스터 샴푸               │  │
│  │     [메인 이미지]     │  │  Purple Booster Shampoo      │  │
│  │                     │  │                              │  │
│  │                     │  │  ⭐ 4.8 (1,234개 리뷰)        │  │
│  │                     │  │                              │  │
│  ├─────────────────────┤  │  ₩32,000  →  ₩25,600        │  │
│  │ [썸네일] [썸네일] ... │  │  첫구매 20% 할인              │  │
│  └─────────────────────┘  │                              │  │
│                           │  용량: 500ml / 1000ml         │  │
│                           │                              │  │
│                           │  수량: [-] 1 [+]              │  │
│                           │                              │  │
│                           │  [장바구니 담기]  [바로 구매]   │  │
│                           │                              │  │
│                           │  💜 찜하기  📤 공유하기        │  │
│                           └────────────────────────────────┘  │
└────────────────────────────────────────────────────────────────┘
```

### 탭 구성 (상세 영역)

| 탭 | 내용 |
|----|------|
| **상품 설명** | 제품 상세 이미지, 성분 스토리, 사용법 |
| **성분 정보** | 전성분, 주요 성분 설명 |
| **리뷰** | 포토리뷰, 텍스트리뷰, 평점 분포 |
| **Q&A** | 상품 문의 |
| **배송/교환** | 배송 안내, 교환/반품 정책 |

### 상품 설명 섹션 구성
1. 핵심 성분 소개 (퍼플콤플렉스)
2. 제품 특징 3가지
3. 사용 전후 비교
4. 사용 방법
5. 추천 대상
6. 함께 쓰면 좋은 제품

### 하단 요소
- 관련 상품 추천
- 최근 본 상품
- 플로팅 구매 버튼 (모바일)

## 5. BRAND 페이지
